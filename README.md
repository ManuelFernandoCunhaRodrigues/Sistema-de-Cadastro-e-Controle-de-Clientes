# Sistema de Cadastro de Clientes

Projeto desenvolvido com HTML, CSS e JavaScript puro para praticar arrays, objetos, funções, programação orientada a objetos, DOM e LocalStorage.

## Funcionalidades

- Cadastro de clientes
- Listagem de clientes
- Busca por nome
- Filtro por status
- Exclusão de clientes
- Resumo com total de clientes
- Salvamento no LocalStorage

## Tecnologias usadas

- HTML
- CSS
- JavaScript

## Conceitos praticados

- Arrays
- Objetos
- Funções
- Classes
- DOM
- Eventos
- LocalStorage

## Como executar

Abra o arquivo `index.html` no navegador ou use a extensão Live Server no VS Code.
